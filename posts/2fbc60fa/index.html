vultr创建服务器

  1. 点击deploy new server

  1. 选择server-location(推荐日本节点)

  1. 选择server-type(我的是ubuntu18.04)

  2. 选择server-size,如果只是小型网站,可以选择$5/月

  3. 然后选择Deploy Now

  4. 然后进入products,等待一会儿服务器status变为running即可-可以通过ping测试

7.点击进入-查看相关信息

宝塔linux官网

ubuntu一键安装脚本,其他系统也可以在里面找到

wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh

如下是vps的详细信息,待会儿会用到

  • 然后进入linux终端(需要sudo)

  • 然后等待安装后能看到url,user,password-因为已经安装过了,所以我只能查找下,可能与第一次安装有所区别,但是没影响
